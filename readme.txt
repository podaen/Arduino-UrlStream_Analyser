## About The Project

There are many great stream libraries and it's is questioning how quick the respons is from the host. However, this is an example to time the data income of a stream between two buffers of 512bytes standaard.

Here's why:
* If you have a stream, you want to know how quick te server send data and how quick the data comes in.
* You should add you http stream address in to URL's and let us know what your result are.
* If you like what you are seen, I'm glad to know. :smile:

In the near futher there will be a measurement of datarate, corsponding with the bandwith of your network.
